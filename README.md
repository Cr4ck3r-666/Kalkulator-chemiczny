# Kalkulator chemiczny

Jest to kalkulator do podstawowych obliczeń chemicznych takich jak masa molowa ,stężenie molowe i procentowe ,równanie Nersta ,pH ,itp. Bazowo jest stworzony pod systemy oparte o jadro Linuxa ale jak bedzie dobrze to powstanie jeszcze wersja na system Windows i jako aplikacja webowa. Czemu taka nazwa? Dla beki.

Kompilacja i uruchomienie:

sudo apt-get install gcc

```bash
sudo apt-get install gcc
g++ kalkulator.cpp -o kalkulator.o
./kalkulator.o
```

# Disclaimer

Why this calculator is in Polish? Becouse there are few chemistry programs on Linux default in Polish languige.
